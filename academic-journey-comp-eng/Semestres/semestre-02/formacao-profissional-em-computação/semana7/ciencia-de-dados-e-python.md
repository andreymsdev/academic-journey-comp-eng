# Formação Profissional em Computação - Ciência de Dado e  Python


### 1. Instalação de Bibliotecas
- Instalação da biblioteca **Arrow**, que facilita a manipulação e formatação de datas e horas em Python.

### 2. Importação e Autenticação
- Importação de bibliotecas:
  - **oauth2lib**: autenticação com OAuth 1 ou OAuth 2.
  - **NumPy**: computação numérica.
  - **Pandas**: análise e manipulação de dados, especialmente tabelas e séries temporais.

### 3. Coleta de Dados da API
- Definição de funções para acessar a API da plataforma de coleta de dados da Univesp.
- Conexão e autenticação com usuário e senha para resgate de token de acesso (autenticação de segundo nível).

### 4. Listagem e Exame de Dispositivos
- Listagem de dispositivos conectados à plataforma, cada um com sensor **dht22**.
- Exemplo de obtenção de dados de um dispositivo específico em São Paulo usando seu GUID.

### 5. Análise e Visualização de Dados
- Criação de tabelas e gráficos a partir dos dados coletados.
- Agrupamento de dados e cálculos estatísticos: contagem, média e desvio padrão.
- Exemplo de gráfico:
  - **Linha azul**: umidade.
  - **Linha vermelha**: temperatura ao longo do tempo.
- Demonstração da capacidade do **Pandas** de ajustar e exibir dados de forma clara.

## Conclusão
A aula reforça a importância de combinar **Python** e **Ciência de Dados** para representar e extrair informações valiosas de dados brutos coletados por sensores IoT.
