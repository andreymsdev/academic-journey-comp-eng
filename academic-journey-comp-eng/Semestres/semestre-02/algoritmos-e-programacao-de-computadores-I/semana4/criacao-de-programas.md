## Semana 4

- **print:** Função usada para exibir informações na tela.

```python
print('Hello World!')
```
- **input:** Função que lê dados digitados pelo usuário e retorna como string.

```python
nome = input('Digite seu nome: ')
print('Olá, {}'.format(nome))
```
- **eval:** Avalia uma expressão em forma de string e retorna o resultado.

```python
x = 2
eval ('x+5')
```
